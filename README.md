# Projeto ProtecaoTela

## Descrição
O projeto ProtecaoTela é uma aplicação destinada a proteger a tela de dispositivos móveis através de tecnologia de bloqueio programático. Ele busca fornecer uma interface intuitiva e um desempenho otimizado para os usuários.

## Funcionalidades
- Bloqueio automático de tela em determinadas situações.
- Configurações personalizáveis para o usuário.
- Integração com sensores do dispositivo para ativação automática.

## Tecnologias Utilizadas
- **Linguagem**: Python, JavaScript
- **Frameworks**: Flask, React
- **Banco de Dados**: PostgreSQL

## Instalação
Para instalar e rodar o projeto localmente, siga os seguintes passos:
1. Clone o repositório: `git clone https://github.com/JoaoDMitzuiama/ProtecaoTela.git`
2. Navegue até o diretório do projeto: `cd ProtecaoTela`
3. Instale as dependências: `pip install -r requirements.txt`
4. Inicie o servidor: `python app.py`

## Uso
Após a instalação, você poderá acessar a aplicação através do endereço `http://localhost:5000`. Siga as instruções na interface para configurar suas preferências.
